# Advanced React To-Do App

A clean, responsive, and modern to-do application built with React, Redux, and Firebase.  
Directly add, delete, and prioritize tasks — no login required for demo!

Live Demo → https://todo-6ooxtshn4-shahins-projects-430e94f7.vercel.app

## Features
- Add & delete tasks with High/Medium/Low priority
- Real-time location-based weather using OpenWeatherMap API  
  (fallback to London if location denied)
- Tasks saved in Firebase Firestore (persists on refresh)
- Fully responsive design (mobile + desktop)
- Clean UI with Bootstrap + custom CSS
- Graceful error handling

## Tech Stack
- **Frontend**: React, Redux Toolkit
- **Styling**: Bootstrap + Custom CSS
- **Database**: Firebase Firestore
- **API**: OpenWeatherMap + Geolocation API
- **Deployment**: Vercel
- **State Management**: Redux + async thunks
